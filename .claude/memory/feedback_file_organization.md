description: Общие ресурсы и информацию размещать в Information/, а не в Files/My/Claude/
type: feedback
---

My/Claude/ — только для кода (классы, индикаторы, EA). Справочные материалы, ссылки, ресурсы — в отдельную папку Information/.

**Why:** Альберт хочет чёткое разделение: Claude-код отдельно, информационные материалы отдельно.

**How to apply:** При создании файлов определять тип контента. Код → My/Claude/ в соответствующей директории. Документация, ссылки, заметки → Information/.
