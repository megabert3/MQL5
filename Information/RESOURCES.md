# Ресурсы по алгоритмической торговле

## Кодовые базы (открытый код)

- [best-of-algorithmic-trading](https://github.com/merovinh/best-of-algorithmic-trading) — рейтинг open-source библиотек, ботов, инструментов. Обновляется еженедельно
- [awesome-systematic-trading](https://github.com/wangzhe3224/awesome-systematic-trading) — огромный каталог: фреймворки, данные, книги. Crypto, Forex, акции, фьючерсы
- [awesome-algorithmic-trading](https://github.com/joelowj/awesome-algorithmic-trading) — библиотеки, ресурсы, учебные материалы
- [geraked/metatrader5](https://github.com/geraked/metatrader5) — готовые EA для MT5 на MQL5 с исходниками
- [GitHub: mql5 topic](https://github.com/topics/mql5) — все MQL5-проекты на GitHub
- [GitHub: algorithmic-trading topic](https://github.com/topics/algorithmic-trading) — 3000+ репозиториев по алготрейдингу

## MQL5 — официальное сообщество

- [MQL5 Code Base](https://www.mql5.com/en/code) — бесплатные EA, индикаторы, скрипты с исходниками
- [MQL5 Articles + Examples](https://www.mql5.com/en/articles/examples) — статьи с кодом: от простых MA до нейросетей
- [MQL5 Articles — Best EA](https://www.mql5.com/en/articles/mt5/examples_experts/best) — лучшие статьи по созданию торговых роботов
- [MQL5 Book](https://www.mql5.com/en/book) — полный учебник по MQL5
- [MQL5 Docs](https://www.mql5.com/en/docs) — справочник языка

## Образование — от основ до продвинутого

- [BabyPips — Chart Patterns](https://www.babypips.com/learn/forex/chart-patterns-schmatterns) — полный курс Forex с нуля, отличные объяснения паттернов
- [Investopedia](https://www.investopedia.com/) — энциклопедия: индикаторы, стратегии, фундаментальный анализ
- [TradingView Scripts](https://www.tradingview.com/scripts/) — тысячи индикаторов и стратегий с открытым кодом (Pine Script)
- [TradingView — Chart Patterns](https://www.tradingview.com/scripts/chartpattern/) — алгоритмическое распознавание паттернов
- [PyQuant News — Guide](https://www.pyquantnews.com/free-python-resources/guide-to-quantitative-trading-strategies-and-backtesting) — гайд по стратегиям и бэктестингу на Python

## Price Action & Smart Money Concepts (SMC)

- [Daily Price Action — SMC Strategy](https://dailypriceaction.com/blog/smc-trading-strategy/) — подробный разбор SMC: структура, ликвидность, Order Blocks
- [HowToTrade — SMC](https://howtotrade.com/trading-strategies/smart-money-concept-smc/) — пошаговая стратегия Smart Money с примерами
- [ICT Trading PDF](https://howtotrade.com/wp-content/uploads/2023/11/ICT-Trading-Strategy-1.pdf) — PDF-гайд по ICT стратегиям (Inner Circle Trader)
- [ICT Concepts — TradeThePool](https://tradethepool.com/technical-skill/ict-concepts/) — обзор концепций ICT: FVG, OB, liquidity sweep
- [InnerCircleTrader.net](https://innercircletrader.net/tutorials/ict-pdf/) — официальные материалы ICT — 4th Edition
- [TradingView — ICT Mastering](https://www.tradingview.com/chart/BTCUSDT.P/XrrOfC4G-Mastering-ICT-Concepts-The-Ultimate-Trading-Strategy-Guide/) — визуальный гайд по ICT на графике BTC

## Книги

- **Encyclopedia of Chart Patterns** — Thomas Bulkowski. Статистика по каждому паттерну: win rate, средний ход, failure rate
- **Technical Analysis of Financial Markets** — John Murphy. Классический учебник, покрывает всё
- **Trading Price Action** (3 книги) — Al Brooks. Глубочайший разбор Price Action, свеча за свечой
- **Quantitative Trading** — Ernest Chan. Как строить и тестировать количественные стратегии

## Научные статьи и количественный анализ

- [SSRN — Futuretesting](https://papers.ssrn.com/sol3/Delivery.cfm/SSRN_ID4728952_code802495.pdf?abstractid=4647103) — проблема переоптимизации и синтетические данные для тестирования
- [NYU — Online Quant Strategies](https://www.stern.nyu.edu/sites/default/files/2025-05/Glucksman_Lahanis.pdf) — академический обзор количественных стратегий
- [Jonathan Kinlay Blog](https://jonathankinlay.com/) — блог квант-трейдера: модели, стратегии, исследования
- [QuantEvolve — arXiv](https://arxiv.org/html/2510.18569v1) — multi-agent AI фреймворк для автоматического создания стратегий
